Nous avons réalisé notre interface avec Bootstrap 5.

Comme indiquer dans le projet, nous qvons juste simuler les différentes interactions sur nos pages Web.
Ces interactions sont les suivants:
1) La page d'accueil
	De cette page nous pouvons soit nous connecter en cliquant sur le lien "Log in" en haut à droite, 
	soit effectuer une recherche grâce au bouton "Boogle Search";
	Si nous nous connectons nous verrons une possibilité de nous déconnecter. En étant connecté, lorsque 
	nous ferons une recherche, il nous sera possible de nous déconnecté aussi en étant sur la page de 
	recherche.

2) La page de connexion
	Nous saisissons un identifiant de connexion et un mot de passe pour pouvoir nous connecter.
	Il est possible aussi d'avoir un autre mot de passe si nous avions oubié le précédent.
	Cependant si nous n'avons pas encore de compte, il suffit de cliquer sur le lien "S'inscrire" en bas
	de la page pour être rédirigé sur la page d'inscription.

3) La page d'inscription
	Sur cette page nous renseignons des afin des bénéficiers d'un compte. 
	Cependant étant sur cette page si nous avons déjà un compte nous pouvons être directement rediriger 
	sur la page de connexion.

4) La page recherché
	Sur la page recherché nous avons la possiblité de nous connecter, de nous déconnecter et de retourner
	sur la page d'accueil.

En somme ce sont les dofférentes intaractions dont nous avons avec nos pages Web.
